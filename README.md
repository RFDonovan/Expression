## Installation
###1-get Git & NodeJS installed on computer

###2-clone this repo
```
$git clone <link>
```
###3-if there is an error module, go to the project folder and launch:
```
$npm install [--proxy: http://myproxy:port]
```



## start

```
$cd Expression
$npm start
```
